# Тесты для лабораторной работы #02 для задания #03

## Входные данные 
Целое n и массив a

## Выходные данные
Изменённый массив a

## Позитивные тесты:
 - 00 - массив не содержит палиндромов
 - 01 - любые числа

## Негативные тесты:
 - 00 - n не число
 - 01 - n ноль
 - 02 - n больше 10
 - 03 - в массиве числа палиндромы
 - 04 - в массив записано не число
