# Тесты для лабораторной работы #04 для задания #02

## Входные данные:
Строка символов

## Выходные данные:
Количество вхождений слова

## Позитивные тесты:
 - 00 - все слова разные
 - 01 - слова одинакового размера
 - 02 - любые слова 

## Негативные тесты:
 - 00 - неверная длина строки
 - 01 - неверная длина слова
 - 02 - пустая строка
