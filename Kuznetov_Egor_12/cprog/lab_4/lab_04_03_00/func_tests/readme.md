# Тесты для лабораторной работы #04 для задания #03

## Входные данные:
Строка символов

## Выходные данные:
Изменённая строка

## Позитивные тесты:
 - 00 - все слова разные
 - 01 - слова состоят из одинаковых символов
 - 02 - любые слова 

## Негативные тесты:
 - 00 - неверная длина строки
 - 01 - неверная длина слова
 - 02 - пустая строка
 - 03 - пустая итоговая строка
